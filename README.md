## Installation & Run (On Android via Termux)
1. **Clone the Repo:**
    ```bash
    git clone <repo-link>
    cd inspireme
    ```

2. **Install Flask:**
    ```bash
    pip install flask
    ```

3. **Run the App:**
    ```bash
    python app.py
    ```

4. Open your browser and go to:
    ```
    http://127.0.0.1:5000/
    ```
